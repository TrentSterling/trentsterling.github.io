Notesdash

Local AI-powered notes dashboard for Windows.
Scan your drives. Analyze with Ollama. Search everything.

View on GitHub Download
🔍

FTS5 Full-Text Search

Search across file paths, AI-generated topics, and summaries instantly with SQLite FTS5 and 300ms debounced input.

🧠

Local AI Analysis

Ollama-powered sentiment analysis, topic extraction, and one-line summaries. Your data never leaves your machine.

⭐

Favorites

Star important notes for quick access. Filter by favorites from the sidebar or toolbar. Persists across rescans.

📊

Topic Distribution

Visual bar chart of topic frequency across all your notes. See what you write about most at a glance.

⚡

Fast Parallel Scanning

jwalk-powered parallel filesystem walk scans thousands of files in seconds. 8000+ file virtual scrolling table.

🛠️

GPU Diagnostics

Live status lights show Ollama connection, model availability, GPU acceleration, and VRAM usage in real time.

Quickstart

# clone and build
git clone https://github.com/TrentSterling/notesdash.git
cd notesdash
cargo run --release

# make sure ollama is running for AI analysis
ollama serve
ollama pull qwen2.5:14b

Requirements

  • ✓ Windows 10/11
  • ✓ Rust toolchain (for building from source)
  • ✓ Ollama running locally (for AI features)
